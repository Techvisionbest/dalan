划重点：知识点

vue-router中的linkActiveClass属性

vue-router(Vue的路由器)组件通过标签，通过其”to”属性指定目标地址，默认渲染成带有正确链接的< a >标签（可以通过配置tag属性渲染成指定标签）。另外，当目标路由成功激活时，链接元素自动设置一个表示激活的css类名，这个类名默认为：”router-link-active”。可以通过路由的构造选项linkActiveClass来全局配置。

flex布局

Flex 是 Flexible Box 的缩写，意为”弹性布局”，用来为盒状模型提供最大的灵活性。

注意

任何容器都可以设置为flex布局，设置为flex布局后，子元素的float、clear和vertical-align属性将失效。

容器的属性：

  • flex-direction：row(默认)|row-reverse|column|column-reverse;子元素的排列方向
  • flex-wrap:nowrap(默认)|wrap|wrap-reverse;子元素的换行情况
  • flex-flow:flex-direction属性flex-wrap属性的简写形式
  • justify-content：flex-start(默认)|flex-end|center|space—between|space-around子元素在主轴上的对齐方式
  • align-items：flex-start|flex-end|center|stretch(默认)|baseline子元素在交叉轴的对其方式
  • align-content：flex-start|flex-end|center|space-between|space-around|stretch(默认);多根轴线的对齐方式

子元素的属性：

  • order：定义子元素在容器中的排列顺序，数值越小，排列越靠前，默认为0
  • flex-grow：定义子元素的放大比例，默认为0，即如果存在剩余空间也不放大
  • flex-shrink：定义子元素的缩小比例，默认为1，即如果空间不做，该子元素将缩小
  • flex-basis：属性定义了在分配多余空间之前，项目占据的主轴空间（main size）。浏览器根据这个属性，计算主轴是否有多余空间。它的默认值为auto，即项目的本来大小。
  • flex:flex-grow, flex-shrink 和 flex-basis的简写，默认值为0 1 auto。后两个属性可选。
  • align-self:允许单个项目有与其他项目不一样的对齐方式，可覆盖align-items属性。默认值为auto，表示继承父元素的align-items属性，如果没有父元素，则等同于stretch。

position:relative的具体作用

生成相对定位，相对于其正常文档流位置定位。
为什么有时候容器需要使用position：relative属性呢？

absolute的英文意思是绝对的意思,实际上是针对父级元素元素定位,如果父级元素没有position:relative|absolute,则追至再上一个父级元素,直至相对于文档的左上角定位，按照我们中国人的理解观念，这个其实是相对定位，是脱离文档流的。用了abolute属性，原有float属性将失效；

relative的英文意思是相对的意思，实际上是相对于对象当前位置的定位。而且是不脱离文档流的，就算用top、lef、bottom、right或margin将其移动位置，它也会在原来的文档流中占有自己实际大小的一块位置。
说白了，absolute是相对于父对象定位，relative是相对于当前位置定位！就是这么简单！

vertical-align属性

vertical-align属性只对inline或者inline-block属性起作用

text-overflow

text-overflow属性规定了当文本溢出包含元素时发生的事情：
text-overflow：clip(默认)|ellipsis|string;
分别代表了：
clip：修剪文本
ellipsis：显示省略号来代表被修剪的文本
string：使用给定的字符串来代表被修剪的文本

CSS中的opacity属性

opacity属性规定元素的不透明级别，文本和背景色都受到不透明级别的影响。

CSS样式：filter：blur(10px)

给图像设置高斯模糊。”radius”一值设定高斯函数的标准差，或者是屏幕上以多少像素融在一起， 所以值越大越模糊；
如果没有设定值，则默认是0；这个参数可设置css长度值，但不接受百分比值。
