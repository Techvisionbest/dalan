一名程序员的总结：2018，追逐更美好的一年

大学刚入学仿佛发生在昨天，时间就像脱缰的野驴，一下子就跑到了大学毕业季。

2017年的形势情

《资治通鉴》荀悦论曰：

夫立策决胜之术，其要有三，一曰形，二曰势，三曰情。
形者，言其大体得失之数也。
势者，言其临时之机进退之宜也。
情者，言其心志可否之实也。
故策同、事等而功殊者，三术不同也。

什么意思呢？

决定一个规划成败的因素主要有三点：

1.形：主流的发展趋势

2.势：临时做出的选择

3.情：心态是否够坚定

2017 年人们讨论的话题很多：大数据、人工智能、区块链、比特币、买房、新零售、内容付费、互联网下半场、工业互联网、实体经济…

这些话题里有的太高端我接触不了，我也分不清究竟是不是昙花一现，我能做的就是尽我自己最大的能力做好自己的事情，尽我所能学到最多的东西。

2017年做的那些选择

人们总说“一念天堂、一念地狱”，无数个选择让我走到现在这个境地，有些是上帝的安排，有些则是自己长期准备的结果。

2017年我做了这些选择：

工作

俗话说女怕嫁错郎、男怕入错行，作为程序员，我们每天奉献大量的时间给工作，如果这份工作不能给予你合理的成长机会，要好好思量下了。

虽然自己从事的是Android开发，但不得不承认的是Android开发越来越不景气了，IOS也好不到哪去，10月份入职的时候一直茫然找不到自己该学习和进步的方向，每天浑浑噩噩的，压力山大。然后经理让我接触了Weex，感谢经理，让我有了大前端的概念。确定了大前端的发展方向，剩下的就是努力了。我很庆幸没有多浪费时间。

感情

2017年12月3日结束了自己的单身生活。

很感谢我的程序媛女朋友大咩咩。

十分期望以及相信我们能一直走下去

2017年心境的转变

程子曰：“君子循理，故常舒泰；小人役于物，故多忧戚。”（见《论语集注》）

君子按照天地的规律要求自己，所以舒适安宁。小人则被外物所奴役，求名逐利，常常斤斤计较，患得患失，以满足自己的私欲。

我常常被外界的物质影响，要么过分骄傲自大、要么羡慕嫉妒，根本原因就是过分关注外界，而没有把精力放到提升自己上。

今年因为这些毛病吃了不少亏，次数多了后，我决定用一张纸写下自己目前的状况，客观地分析了自己做这些事情的投入、收益的成本和概率，又请教了一些前辈，最后决定还是稳妥起见，在快速成长的阶段先专注提升自己，然后再去尝试各种机会。

2018年要做的

2017 结束的我，仍然没有房没有车没有钱，但与去年相比，所处环境和心境有了很大的变化。
翻了翻去年的总结和今年的一些记录，感慨颇多,要做的也很多。

开始注重健康

前段时间朋友圈风传的一篇关于 90 后开始养生的文章，我觉得能反映出一个问题：90 后普遍觉得自己还能抗能打，现实狠狠地抽了他们一个巴掌。

在我们心里觉得自己还是“祖国的花朵”、“早晨八九点钟的太阳”，结果上体重秤后才发现；竟然已经这么胖了，体检后才发现，竟然有这么多毛病了；计算下年龄才发现，竟然已经要奔三了。

老得太快。

以前总觉得“祝你身体健康”是件多么无聊的祝福，等到自己身体有些不舒服了，才觉得这句话的诚恳。

今年明显感觉我的身体不如以前了，视力退步、颈椎酸痛 、经常腰酸背痛，长期熬夜、伏案造成的伤害开始逐渐反噬了。

2018年，一定要开始注重身体健康了。

Github

从我注册Github到现在没有commit过一个项目，也从来没有提过任何的issues，Github对我来说更像是copy别人代码的工具。
觉得自己从来没有充分利用过这个工具。
2018年，要重度使用Github，commit项目，提issue，让Github活跃起来。
